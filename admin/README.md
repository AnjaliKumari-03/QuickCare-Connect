# 🛠 QuickCare Admin Panel

This is the admin dashboard for **QuickCare Connect**, built using React (Vite).  
It allows administrators and doctors to manage appointments and system data.

---

## 🚀 Features

### Admin
- Manage doctors
- View all appointments
- Monitor system activity

### Doctor
- View assigned appointments
- Manage availability
- Track patient details

---

## 🧰 Tech Stack

- React.js (Vite)
- Tailwind CSS
- Axios
- Context API
